🔐 UPI Fraud Detection System

A system that detects suspicious UPI transactions using rule-based analysis to prevent fraud in real-time.

🚀 Live Demo
🌐 Frontend: https://upi-fraud-detection-system.vercel.app/

🧠 Problem
UPI fraud cases are increasing due to lack of real-time detection and user awareness.

💡 Solution
Detects fraud based on:
High transaction amount
Repeated transactions
Unknown receiver

✨ Features
🔍 Real-time detection
🚨 Fraud alerts
📊 Pattern analysis

🛠️ Tech Stack
Language: Html/css/js/Python
Storage: File / DB
Logic: Rule-based

⚙️ How It Works
Input transaction
Analyze conditions
If suspicious → 🚨 Alert
